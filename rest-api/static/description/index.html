What is REST API?

REST API (Representational State Transfer) is a standard way for applications to communicate over HTTP.

It allows your Odoo system to act as a backend service that can be accessed from anywhere.

What is Swagger?

Swagger is an interactive API documentation tool that allows developers to test APIs directly in the browser.

  • Explore endpoints visually
  • Test API without external tools
  • Understand request & response instantly

API Endpoints

Authentication

  • POST /api/v1/login
  • POST /api/v1/logout

Products

  • GET /api/v1/products
  • POST /api/v1/products
  • PUT /api/v1/products/{id}
  • DELETE /api/v1/products/{id}

Authentication Flow

  1. Login and receive JWT token
  2. Include token in header
  3. Access protected endpoints
  4. Logout when finished
Authorization: Bearer your_token_here

Installation

  1. Copy module into addons folder
  2. Update Apps List
  3. Install the module

Requirements

Install required Python library:

pip install PyJWT
